# XTIANZ.com GitHub Pages Website

This folder is ready for GitHub Pages.

## Upload to GitHub

1. Go to GitHub and create a public repository named `xtianz.com` under username `lomaximo888`.
2. Upload these files and folders to the repository root:
   - `index.html`
   - `CNAME`
   - `assets/`
   - `README.md`
3. Commit changes.
4. Go to repository **Settings → Pages**.
5. Set **Source** to `Deploy from a branch`.
6. Set **Branch** to `main` and folder to `/root`.
7. Save.

Your temporary GitHub Pages URL should be:

https://lomaximo888.github.io/xtianz.com/

## Custom domain

The included `CNAME` file already points to:

xtianz.com

In GitHub Pages custom domain settings, also enter:

xtianz.com

Then configure Bluehost DNS:

A @ 185.199.108.153
A @ 185.199.109.153
A @ 185.199.110.153
A @ 185.199.111.153
CNAME www lomaximo888.github.io

After DNS is verified, enable **Enforce HTTPS**.
