## Upload to GitHub

1. Extract the ZIP on your computer.
2. Go to your repository: `https://github.com/lomaximo888/xtianz.com`
3. Click **Add file → Upload files**.
4. Drag all extracted files and folders into GitHub.
5. Replace existing files if GitHub asks.
6. Commit with:

```text
Deploy XTIANZ v9
```

## GitHub Pages settings

Go to:

```text
Settings → Pages
```

Use:

```text
Source: Deploy from branch
Branch: main
Folder: /(root)
Custom domain: xtianz.com
```

Make sure **Enforce HTTPS** is enabled after DNS is verified.
