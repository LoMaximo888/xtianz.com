# XTIANZ v32 — AI Signal Lab + DMV Pulse

This static GitHub Pages build adds a redesigned AI-first homepage, a curated DMV/LoCo source navigator, and an original guide connecting Loudoun County infrastructure with the AI stack.

## New
- `dmv.html` — DMV / LoCo local-news source navigator
- `articles/dmv-ai-infrastructure-corridor.html` — original long-form guide
- Updated global navigation and command search
- Redesigned homepage hero, signal console, local source cards, and regional visual system
- Direct links to The Burn, Loudoun Now, Loudoun County official alerts, WTOP, FFXnow, ARLnow, NBC4, and WUSA9

## Editorial behavior
- XTIANZ links to original local publishers; it does not copy their reporting.
- The AdSense loader remains in the `<head>`, but there are no visible ad units in this review-oriented build.
- Local source links open the latest publisher pages, which keeps the navigator useful without fragile scraping or copied headlines.
